# dyx-blog 项目进度文档

## 1. 项目概述

- 项目名称：dyx-blog
- 项目定位：个人品牌主页 + 求职简历展示 + 博客内容站 + 后台管理系统
- 前端技术：Vue 3、TypeScript、Pinia、Vue Router、Axios、Tailwind CSS、Element Plus
- 后端技术：Spring Boot 3、MyBatis-Plus、MySQL、JWT、本地媒体存储
- 管理方式：前台与后台共用一个前端项目，通过 `/dyx-manager` 路由分区

## 2. 当前功能模块划分

### 前台模块

1. 首页
