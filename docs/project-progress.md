# dyx-blog 项目进度文档

## 1. 项目概述

- 项目名称：dyx-blog
- 项目定位：个人品牌主页 + 求职简历展示 + 博客内容站 + 后台管理系统
- 前端技术：Vue 3、TypeScript、Pinia、Vue Router、Axios、Tailwind CSS、Element Plus
- 后端技术：Spring Boot 3、MyBatis-Plus、MySQL、JWT、本地媒体存储
- 管理方式：前台与后台共用一个前端项目，通过 `/admin` 路由分区

## 2. 当前功能模块划分

### 前台模块

1. 首页
2. 关于我（个人资料 + 个人作品 + 荣誉时间线）
3. 简历（含 PDF 下载与打印）
4. 个人最新动态
5. 博客列表
6. 博客详情

### 后台模块

1. 登录认证
2. 仪表盘
3. 访问日志
4. 博客管理
5. 动态管理
6. 荣誉管理
7. 关于我管理
8. 简历管理
9. 作品管理
10. 媒体资源管理
11. 用户管理

- About 页面使用 `Promise.allSettled` 拉取资料、作品与荣誉，降低单接口失败时的空白风险
