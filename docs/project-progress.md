# dyx-blog 项目进度文档

## 1. 项目概述
- 项目名称：dyx-blog
- 项目定位：个人博客网站 + 个人展示站 + 后台管理系统
- 前端技术：Vue 3、TypeScript、Pinia、Vue Router、Axios、Tailwind CSS、Element Plus
- 后端技术：Spring Boot 3、MyBatis-Plus、MySQL、JWT、本地媒体存储
- 管理方式：前台与后台共用一个前端项目，通过 `/admin` 路由分区

## 2. 功能模块划分
### 前台模块
1. 首页
2. 个人最新动态
3. 项目经历
4. 个人照片
5. 个人经历
6. 个人信息
7. 博客列表
8. 博客详情

### 后台模块
1. 登录认证
2. 仪表盘
3. 博客管理
4. 动态管理
5. 项目经历管理
6. 照片管理
7. 个人资料管理
8. 媒体资源管理
9. 用户管理
